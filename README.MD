#detectbook
To run please:
1) Have docker and docker-compose at your computer
2) Run `docker-compose up -d`
3) Check that localhost is running with http://0.0.0.0:8100/
4) Go to the react folder `cd detectbook`
5) `npm install`
6) `npm run`
7) A window should open with the application.
